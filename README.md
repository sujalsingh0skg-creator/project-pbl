# 📌 Overview

DocuSense AI is a full-stack enterprise invoice intelligence platform that automates invoice extraction, validation, analytics, and reporting using Microsoft Azure AI Document Intelligence.

The system allows users to upload invoice PDFs or images, extract important financial fields using AI, store records in SQLite, generate analytics dashboards, and export professional PDF reports.

# ✨ Features

## 🔐 Authentication System
- Secure Flask-based login
- Session management
- Protected API routes

## 📄 Smart Invoice Processing
- Upload PDF and image invoices
- Drag & Drop invoice intake
- Multi-invoice batch analysis
- Azure AI-powered extraction

## 🧠 AI Field Extraction
Extracts:
- Invoice Number
- Vendor Name
- Invoice Date
- Tax Amount
- Total Amount Due

## 📊 Analytics Dashboard
- Real-time confidence metrics
- Validation accuracy monitoring
- Processing time analytics
- Interactive Chart.js visualizations

## 🗄️ Database Integration
- SQLite invoice history storage

## 📥 Export System
- PDF report generation

## 🎨 Premium UI/UX
- Responsive design
- Dark/Light theme
- Modern glassmorphism interface

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/DocuSense-AI.git
cd DocuSense-AI
```

---

## 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 3️⃣ Run Flask Server

```bash
python app.py
```

---

## 4️⃣ Open in Browser
